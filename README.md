## 빌드 명령어

```cli
npm run dev : 개발 빌드
npm run build : 배포 빌드용
```

## npm 버전 관리 관련

```cli
npm install -g npm-check-updates
ncu -u
```

## 실행 방법 가이드  
[Node.js](https://nodejs.org/ko) 다운로드 및 설치   
<p>각 섹션 터미널에서</p>  

```cli
npm install
```

<p>설치 완료 이후</p>

```cli
npm run dev
```

<p>이후 터미널 창의 localhost 의 개발용 링크 타고 웹 브라우저를 열어서 확인</p>
